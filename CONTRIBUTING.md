# 🤝 Contributing to awesome-football

## How the project accepts contributions

Per the README: **"Contributions welcome. Anything missing? Send in a pull request. Thanks."**

### Contribution types welcomed
- **Datasets** — Open data in CSV, JSON, etc.
- **Apps** — Open source tools for scores, picks, predictions, office pools
- **Technology** — Libraries, APIs, scrapers
- **Cultural / Documentation content** — Match day culture, fan experiences, community discussions
- **Fixes** — Typos, broken links, outdated data

### Contribution guidelines
1. **Fork the repo** and create a new branch
2. **Submit a PR** with clear title and description
3. **Cite your sources** — All data should reference its origin
4. **Keep content current** — Review and update annually
5. **Be inclusive** — Cover all tiers of the football pyramid
6. **Public domain dedication** — Content contributed is dedicated to the public domain

### For cultural / documentation content
- Use the existing section naming convention (⚽ prefix for football-specific sections)
- Reference the `NON-LEAGUE-MATCHDAY-CULTURE.md` for comprehensive match day culture research
- Link to community discussion platforms and forums
- Include fan quotes and sentiment data alongside statistics
- Add cost comparisons and practical away day guides
